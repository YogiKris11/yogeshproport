# **App Name**: AI Engineer Portfolio

## Core Features:

- Hero Canvas Animation: Implement a scroll-triggered, frame-by-frame canvas animation on the hero section, featuring the glass monolith and system expansion concept with smooth physics-based interactions.
- Featured Projects Section: Showcase AI-powered apps, full-stack systems, and system architecture builds with project name, tech stack tags, system description, 'View Case Study' CTA, and links to GitHub/Live Demo.
- Engineering Philosophy Section: Present a split layout featuring structured content blocks (System Thinking, Performance-First Engineering, AI Integration, Scalable Architecture) and an animated geometric system visualization.
- Tech Stack Section: Display an elegant grid of AI/ML, Full Stack, DevOps, Systems, and Tooling technologies with animated glowing outlines on hover.
- Final Call-To-Action: Include a minimal, powerful CTA with a title ('Let's Build Intelligent Systems.') and a button ('Start a Conversation') on a deep gradient background with a subtle rotating glow ring.
- AI-Powered Project Descriptions: Implement a tool to provide suggestions for project descriptions using generative AI, ensuring they are system-level and concise.

## Style Guidelines:

- Primary background color: Deep gray-blue (#0B0F14) for a premium tech feel.
- Secondary background color: Dark slate gray (#111827) to create subtle depth.
- Accent color: Electric Blue (#3B82F6) for interactive elements and highlights.
- Headings: 'Space Grotesk' (sans-serif) for a computerized feel.
- Body: 'Inter' (sans-serif) for a modern and neutral look.
- Use minimal, professional icons for project links and tech stack sections.
- Implement smooth spring interpolations and subtle animations for hover effects and transitions.
